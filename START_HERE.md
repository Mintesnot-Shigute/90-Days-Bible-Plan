## 🚀 What's Next (15 Minutes)

### Step 1: Create Supabase Project (5 min)
```
1. Go to supabase.com
2. Create new project
3. Copy Project URL and Anon Key
4. Run supabase/schema.sql in SQL Editor
```

### Step 2: Update .env File (1 min)
```
Edit .env and add your Supabase credentials:
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```

### Step 3: Test Locally (2 min)
```powershell
npm run dev
# Open http://localhost:5173
# Test by entering a name and checking off a reading
```

### Step 4: Push to GitHub (3 min)
```powershell
git add .
git commit -m "90-day Bible tracker"
git push origin main
```

### Step 5: Deploy to Vercel (3 min)
```powershell
vercel --prod
# OR use Vercel dashboard
```

### Step 6: Share Link (1 min)
```
Send your Vercel URL to 8 friends
They click → enter name → start reading together
```

**Total: ~15 minutes to live**
